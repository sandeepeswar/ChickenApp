### Loop

```java
class LoopDemo{
	public static void main(String []cmdArgs){
		int i, j=1;
		//for loop
		for(i=1; i<=10; i++){
			System.out.println(i);
		}
		
		//while loop
		while(j<=10){
			System.out.println(j);
			j++;
		}
		
		//foreach loop
		int []arr = {1, 5, 7, 8, 9, 10};
		for(int item:arr){
			System.out.println(item);
		}
	}
}

```
